# 📊 Animal Shelter Data Analysis | Turning Data into Impact

I’m excited to share my latest Animal Shelter Data Analysis Dashboard, where I explored animal intake, outcomes, and overall shelter performance using interactive visualizations. This project provides a clear overview of key metrics such as total animals, intake trends, adoption outcomes, and survival rates.

📌 Key Insights:

- A significant portion of animals fall under stray intake, highlighting the need for better awareness and control measures
- Adoption and rescue outcomes contribute to the majority of positive outcomes, while a notable number of cases still result in deaths
- Monthly trends show peak intake and outcome activity during mid-year periods
- Most animals are categorized under normal health conditions, with fewer cases of injury or illness
- Dogs and cats dominate shelter populations, with varied outcomes across categories

🛠️ Tools & Technologies Used:

- Power BI (Dashboard design & interactive visualization)
- Microsoft Excel (Data cleaning & preprocessing)
- DAX (Calculated measures & KPIs)
- Data Modeling (Relationships & performance optimization)

This project enhanced my ability to analyze real-world datasets and present meaningful insights that can support better decision-making in animal welfare initiatives.
